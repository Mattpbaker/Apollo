## How to Get LinkedIn Profile Pictures

1. Visit each LinkedIn profile
2. Right-click on the profile picture
3. Select "Save Image As" or "Copy Image"
4. Save with the correct filename in this folder
5. Supported formats: `.jpg`, `.jpeg`, `.png`

The component will automatically display the images once they're added. If an image is missing, it will show a gray placeholder circle.
